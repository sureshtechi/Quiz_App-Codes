About Us

This Web Application for conducting and attending quiz was a 3rd year project in our college. With the help of our professor and friends we were able to successfully finish our project. We hope it is simple and usefull for all our users for both conducting and attending the quizzes.

Our Services

Create Quizzes

You can create quizzes in different style, it can be linear or non-linear, MCQ or Descriptive, Single correct or Multi correct, etc. Each quiz has a unique code which you can give to users who want to attend the quiz.

Create Quiz

Attend Quizzes

You can attend quizzes with the code which is provided by the user who is conducting the quiz.there is a option to check your answer after the Quiz timing is over.You can also upload images of answer if enabled by quiz creater.

Attend Quiz

Track your Activity easily

You can track your activities easily with the user friendly UI. The Application keeps track of all the quizzes you have conducted and user who have attended the quiz with detailed response. It also has the details of all the quizes you have attended.

Track Activity
